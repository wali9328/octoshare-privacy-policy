Email address, hashed password, referral identifiers.

Device model, OS version, IP address, network type, device identifiers, app version.

2.3 Bandwidth Usage Data

Bandwidth shared, connection quality, session duration, throughput metrics.

Earnings balance, referral earnings, payment history, payout details.
